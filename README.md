InsightIQ
A Streamlit web app that analyzes customer reviews using Hugging Face sentiment analysis and turns raw text into useful business insights. It is designed to help users upload review datasets, run sentiment analysis in batches, and view results quickly in an interactive interface.

Features
Upload CSV datasets with customer reviews.

Analyze sentiment using a Hugging Face transformer model.

Process reviews in batches for better performance.

Display sentiment labels and confidence scores.

Built with Streamlit for a simple and interactive UI.

Uses caching to improve speed on repeated runs.
